## 🎯 核心任务

### 1. 🔴 P0: 简体中文本地化（100%覆盖）

**目标**: 所有界面文本支持简体中文，无硬编码英文

**文件范围**:
- `app/page.tsx`
- `app/models/page.tsx`
- `app/sessions/page.tsx`
- `app/skills/page.tsx`
- `app/alerts/page.tsx`
- `app/pixel-office/page.tsx`
- `app/stats/page.tsx`
- `app/components/*`
- `app/sidebar.tsx`

**任务清单**:
- [ ] 扫描所有 `.tsx`/`.ts` 文件
  - [ ] 使用 grep/rg 查找硬编码英文文本
  - [ ] 识别用户可见的文本
- [ ] 在 `lib/i18n.tsx` 补充翻译 key
  - [ ] 为每个硬编码文本添加 key
  - [ ] 翻译为简体中文
  - [ ] 确保繁体中文和英文也有对应翻译
- [ ] 替换硬编码文本为 `t()` 调用
  - [ ] 导入 `useTranslation` hook
  - [ ] 替换硬编码字符串为 `t('key')`
- [ ] 验证 100% 中文界面
  - [ ] 检查每个页面
  - [ ] 测试语言切换
- [ ] 处理动态文本
  - [ ] 参数化翻译
  - [ ] 复数处理

**示例**:
```tsx
// 修改前
<button>Submit</button>

// 修改后
const { t } = useTranslation();
<button>{t('common.submit')}</button>
```
